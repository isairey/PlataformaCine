# 🌌 Acerca del proyecto

**CineGeek** es una plataforma web enfocada en la exploración de películas, series y contenido multimedia moderno.

El proyecto está diseñado para ofrecer información actualizada sobre:

- 🎬 Películas populares
- 📺 Series destacadas
- 🔥 Contenido trending
- 🌎 Cine regional
- ⭐ Rankings multimedia
- 🎞️ Estrenos recientes

CineGeek busca crear una experiencia inmersiva para cinéfilos y fanáticos de las series mediante una interfaz moderna y dinámica.

# ⚖️ Aviso legal

## 📌 Sobre el contenido multimedia

CineGeek:

- ❌ No almacena archivos multimedia
- ❌ No hospeda películas o series
- ✅ Solo enlaza servicios externos
- 🌐 Utiliza proveedores de terceros

Cualquier problema legal relacionado con contenido multimedia debe tratarse directamente con los proveedores externos.
